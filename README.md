💻 Installation
macOS
Download the latest release from the Releases tab
Open the .dmg file
Drag LucaSlicer into your Applications folder
Launch the app
⚠️ If you get a security warning:
Go to System Settings → Privacy & Security
Click Open Anyway
First Launch Setup
Open LucaSlicer
Select your printer (or import included profiles)
Choose a filament preset
Slice your first model 🎉

🧾 Notes
LucaSlicer is based on OrcaSlicer
Some features may still reference OrcaSlicer internally
Recommended workflow for FlashForge users:
Slice in LucaSlicer
